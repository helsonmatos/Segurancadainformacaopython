1 - O que é a Biblioteca Socket?
- Esta biblioteca fornece acesso de baixo nível à interface de rede.

- OS. O fornece a API socket que relaciona o programa com a rede

2 - Desenvolvimento de um cliente TCP (Integridade)
- O TCP (Transmission Control Protocol) ou Protocolo
de Controle de Transmissão é um dos protocolos de comunicação, que dão
suporte a rede global Internet, verificando se os dados são enviados na sequência
correta e sem erros.

- Nosso programa verificará se dados são enviados de maneira íntegra

3 - Desenvolvimento de um cliente UDP (Disponibilidade)
- O UDP (User Datagram Protocol) ou Protocolo de Datagrama de Usuário 
é um protocolo simples da camada de transporte que permite que a aplicação
envie um datagrama dentro num pacote IPv4 ou IPv6 a um destino, porém sem qualquer tipo
de garantia que o pacote chegue corretamente.
