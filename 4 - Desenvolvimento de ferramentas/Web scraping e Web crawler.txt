1 - Web Scraping
- Um web scraper é uma ferramenta de coleta de dados web,
uma forma de mineração que permite a extração de dados de sites
da web convertendo-os em informação estruturada para
posterior análise.

- BeautifulSoup - é uma biblioteca de extração de dados de
arquivos HTML e XML.

- Requests - permite que você envie solicitação HTTP

2 - Web Crawler

de um site. È como um robô que captura informações de cada um dos links que encontra
pela frente, cadastra e compreende o que é mais relevante. (palavras-chaves)
